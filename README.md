# 📚 LMS Management System

A modern Learning Management System (LMS) built with ASP.NET Core MVC following a clean layered architecture.

## 🚀 Live Demo

(https://mohamed12ali-001-site1.dtempurl.com/)

---

## 📌 Features

### 👨‍💼 Admin

- Dashboard with statistics
- Manage Courses
- Manage Categories
- Manage Instructors
- Manage Students
- Manage Enrollments
- Manage Reviews

### 👨‍🎓 Student

- Browse Courses
- Search Courses
- Filter Courses
- View Course Details
- Enroll in Courses
- Cancel Enrollment
- Edit Profile

### 🔐 Authentication

- Register
- Login
- Role-based Authorization

---

## 🛠 Technologies

- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- LINQ
- Repository Pattern
- Service Layer
- Bootstrap 5
- Identity Authentication
- Dependency Injection

---

## 📂 Project Structure

```
Presentation (MVC)

Business Logic Layer (BLL)

Data Access Layer (DAL)

Database (SQL Server)
```
